Repository

A repository is a special folder that stores your code and project files. It can have files, folders, and even other repositories inside it. Think of it as a container that keeps everything for your project in one place.

Software is installed on your system, but Git tracks changes in a specific folder. To check the current state of your repository, run this command:

git rm -r .
